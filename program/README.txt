הוראות הפעלה
picsToPdf.jar

התוכנה מעבירה תקיית תמונות לקובץ pdf, כאשר כל תמונה מתכווצת/ מתרחבת ל95X95mm, ובכל קובץ 18 תמונות.
יש לפתוח את קובץ הjar ולבחור את תיקיית התמונות.
לאחר מכן למלא את השדות (אופציונלי) טקסט, גודל (ברירת המחדל: 15) ושם פונט (Arial), ומרחק מהתחתית (0).
בלחיצה על הפעל יווצרו קבצי הpdf בתיקייה הנוכחית - התקייה של התוכנה (המיקום של קובץ הjar).

ע"מ להפעיל את התוכנה, יש להתקין java על המחשב.
ניתן להתקין את java בקישור: https://java.com/en/download/win10.jsp
